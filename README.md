# 2026-02-KIT
# Mehr als ein Testergebnis  
## Konfidenzintervalle im Unterricht erlebbar machen

Vortrag am KIT, Februar 2026

Dieses Repository enthält Materialien zu einem didaktischen Vortrag
über Konfidenzintervalle im Stochastikunterricht.

Im Mittelpunkt steht die klare Trennung von
Modell und Realität sowie das Verständnis statistischer Verfahren
als modellbasierte Konstruktionen – nicht als Rezepte.

## Inhalt

- **scr/Python/**  
  Modularer Python-Code zur Simulation und Visualisierung  
  von Konfidenzintervallen (core / plot / run).

- **notebooks/**  
  Jupyter-Notebooks für Exploration, Unterricht und Fortbildung.

- **fig/**  
  Automatisch erzeugte Grafiken (z. B. Konfidenzellipsen).

- **docs/**  
  Handreichungen, Designregeln und begleitende Materialien.

## Didaktische Leitidee

Konfidenzintervalle werden nicht „freigerubbelt“,  
sondern ausgehend von Beobachtung, Modell und Wiederholung
schrittweise entwickelt und interpretiert.

Formeln beschreiben das Verfahren –  
sie sind nicht sein Ursprung.

## Hinweise

- Die Simulationen arbeiten im $B(n;p)$-Modell.
- Wiederholungen finden ausschließlich im Modell statt.
- Der Code ist bewusst modular aufgebaut, um Struktur sichtbar zu machen.

## Nutzung

Die lauffähigen Beispiele sind als Jupyter-Notebooks im Ordner `notebooks/` abgelegt.
Sie dienen als Einstiegspunkt für Exploration und Experimente.

Der Python-Code in `scr/Python/` ist modular aufgebaut und wird von den Notebooks verwendet.
